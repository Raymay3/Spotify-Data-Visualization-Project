# Spotify Data Visualization Dashboard (2025) 

• Developed an interactive data visualization dashboard using Python and Plotly to analyze factors 
influencing song popularity across 1,000+ tracks and 10 genres 

• Designed multiple visualizations (scatter, histogram, violin plots) to uncover relationships between 
key audio features (energy, tempo, valence, danceability) 

• Implemented filtering and preprocessing (feature selection, missing data handling) to enable real
time comparison and improve analytical accuracy
